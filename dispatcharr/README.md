# Home Assistant Add-on: Dispatcharr

[Dispatcharr](https://github.com/dispatcharr/dispatcharr) is een open-source beheerplatform voor IPTV-streams, EPG-data en VOD-content — in de stijl van de *arr-familie (Sonarr/Radarr).

Deze add-on draait de officiële all-in-one Dispatcharr-image: web-app, PostgreSQL, Redis en Celery in één container.

## Functies

- Streams proxyen en samenvoegen uit meerdere bronnen
- M3U- en Xtream Codes-ondersteuning
- Automatische EPG-koppeling
- VOD-streaming en realtime monitoring
- Gebruikersbeheer en pluginsysteem

## Snel starten

1. Installeer de add-on.
2. Start de add-on en wacht tot de eerste initialisatie klaar is (database-setup kan bij de eerste start even duren).
3. Klik op **Open web-UI** (poort 9191) en maak je beheerdersaccount aan.

Zie het tabblad **Documentatie** voor alle configuratie-opties.
